# Calm Breathing Routine

A calming breathing app with guided steps, adjustable timing, multiple ambient sounds, and a mobile-friendly layout.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

### Option 1: GitHub import
1. Create a new GitHub repository.
2. Push this project to that repository.
3. In Vercel, click **Add New > Project**.
4. Import the GitHub repository.
5. Vercel should detect **Vite** automatically.
6. Click **Deploy**.

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel
```

Then follow the prompts.
